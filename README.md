1.	Встановлення середовища: Завантажити та встановити пакет XAMPP. Рекомендується встановлювати в корінь диску (наприклад, C:\xampp), щоб уникнути проблем з правами доступу Windows UAC.
2.	Запуск серверів: Відкрити XAMPP Control Panel та запустити модулі Apache та MySQL, натиснувши кнопки "Start". Індикатори повинні стати зеленими, що свідчить про успішний запуск.
3.	Розгортання файлів: Скопіювати папку з файлами веб-сайту (вихідний код) у директорію C:\xampp\htdocs\Deves.
4.	Налаштування бази даних:
•	Відкрити браузер та перейти за адресою http://localhost/phpmyadmin.
•	Створити нову базу даних з назвою, що відповідає налаштуванням у файлі конфігурації. Важливо обрати кодування utf8_general_ci або utf8mb4_unicode_ci для коректного відображення кирилиці.
•	Вибрати вкладку "Імпорт" та завантажити файл дампу бази даних (deves_medical_centre.sql), що постачається разом з проєктом. Натиснути "Go" для виконання SQL-скрипта створення таблиць.
5.	Конфігурація підключення: Перевірити файл налаштувань підключення до БД (зазвичай db_connect.php або config.php) у папці проєкту. Переконатися, що параметри (хост, ім'я користувача, пароль, назва БД) відповідають локальним налаштуванням (за замовчуванням у XAMPP: user=root, password=[пусто]).
6.	Перевірка роботи: Відкрити браузер та перейти за адресою http://localhost/Deves. Головна сторінка сайту повинна завантажитися коректно.
